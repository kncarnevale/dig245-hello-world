Learn-Computing Activities

File Organization and Naming

In the File Organization and Naming learn-computing activity, I did, I learned how and why one would need a naming system to keep all their files organized. Some of the best naming practices are names that are descriptive and avoid abbreviations. When you make a new version of the same file, you must append a number or date to it, either using hyphens, underscores, or camelCase, so it is easier to find the version you are looking for. It is also good practice to structure all your folders in an organized way, so you never have any problems finding a file.

The Command Line

The other learn-computing activity I did was The Command Line. In this activity, I learned that the command line is a text interface in the computer. I also learned how to use the command line to navigate through directories and make new files in them.
